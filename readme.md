# Palindromo index

Retorna el indice de un caracter de la cadena que al removerse convertiría la cadena en un palindromo.

En el caso que ya sea un palindromo o no tenga solución retorna -1.

## Ejemplos:  

    I: apa
    O: -1

    I: array
    O: 4

    I: horno
    O: -1

    I: adfada
    O: 2
