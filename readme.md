# Palíndromo index

Retorna el índice de un caracter de la cadena que al removerse convertiría la cadena en un palíndromo.

En el caso que ya sea un palíndromo o no tenga solución retorna -1.

## Ejemplos:  

    I: apa
    O: -1

    I: array
    O: 4

    I: horno
    O: -1

    I: adfada
    O: 2
